# docbook2md(5) #

# NAME #

docbook2md - Convert Docbook articles and refentry pages to
    Markdown

# SYNOPSIS #

`docbook2md file.xml`

# Description #

docbook2md is an alias for the  `docbook-convert`  utility pre-configured to produce Markdown output \(i.e. Identical to using `docbook-convert --markdown` ). See the `docbook-convert`  man page for the full option list.
