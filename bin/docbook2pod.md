# docbook2pod(5) #

# NAME #

docbook2pod - Convert Docbook articles and refentry pages to Perl
    POD

# SYNOPSIS #

`docbook2pod file.xml`

# Description #

docbook2pod is an alias for the  `docbook-convert`  utility pre-configured to produce POD output \(i.e. Identical to using `docbook-convert --pod` ). See the `docbook-convert`  man page for the full option list.
